# Education Data Analysis

## Project Overview

This project analyzes global education-related data using Python. The dataset contains information about population census, agricultural census, national accounts, and other educational statistics.

## Tools and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

## Dataset

* EdStatsCountry.csv

## Analysis Performed

* Data Cleaning
* Handling Missing Values
* Descriptive Statistics
* Correlation Analysis
* Data Visualization
* Pair Plot Analysis
* Heatmap Visualization

## Visualizations

The project includes various charts such as:

* Bar Charts
* Pie Charts
* Scatter Plots
* Violin Plots
* Correlation Heatmaps
* Pair Plots

## How to Run

1. Download the project files.
2. Install the required libraries:
   pip install -r requirements.txt
3. Open the Jupyter Notebook.
4. Run all cells to reproduce the analysis.

## Conclusion

The analysis provides insights into educational and demographic indicators across different countries and demonstrates the use of Python for data analysis and visualization.
